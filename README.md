# MeBo - Meubles en Métal et Bois

Site web pour l'entreprise MeBo spécialisée dans la fabrication de meubles combinant métal et bois.

## Description

MeBo propose une gamme exclusive de meubles design qui marie la robustesse du métal à la chaleur du bois. Cette application web présente nos créations, notre philosophie et permet aux clients de découvrir nos réalisations.

## Structure du projet

- `public/` - Fichiers accessibles publiquement
  - `index.html` - Page d'accueil
  - `assets/` - Images, CSS et JS publics
- `src/` - Code source de l'application
  - `components/` - Composants réutilisables
  - `pages/` - Pages de l'application
  - `styles/` - Feuilles de style
  - `js/` - Scripts JavaScript
  - `data/` - Données statiques
- `docs/` - Documentation
- `tests/` - Tests de l'application

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (ES6+)
- JSON (pour les données)

## Installation

1. Clonez ce dépôt
2. Ouvrez `public/index.html` dans votre navigateur

## Développement

Pour contribuer au projet :

1. Modifiez les fichiers dans le dossier `src/`
2. Compilez ou copiez les ressources vers `public/` si nécessaire
